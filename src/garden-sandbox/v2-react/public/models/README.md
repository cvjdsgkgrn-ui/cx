# 3D 模型下载清单

将下载的 .glb 或 .gltf 文件放到这个文件夹里，告诉我文件名，我帮你集成到场景中。

---

## 推荐下载源（免费 · CC0 · glTF 格式）

### 1. Poly Pizza — 单体模型，直接下载 glTF
网址: https://poly.pizza

搜索关键词（在网站顶部搜索框输入）:
  - "pagoda"      → 宝塔/亭子
  - "chinese"     → 中国风建筑
  - "japanese gate" → 门/月洞门
  - "bonsai"      → 盆景树
  - "stone lantern" → 石灯
  - "wooden bridge" → 木桥
  - "rock"        → 石头
  - "tree"        → 树

下载方式: 点开模型 → 点击 "Download" → 选 "glTF" 格式

---

### 2. Quaternius — 整套低多边形模型包
网址: https://quaternius.com

推荐下载:
  - "Ultimate Low Poly Pack" (含树木、石头、花草、建筑)
  - "Fantasy Pack"
  - "Nature Pack"

下载方式: 点击 Pack → 下载 .zip → 解压后找 .glb 文件

---

### 3. Kenney — Castle Kit（城堡/围墙/塔楼）
网址: https://kenney.nl/assets/castle-kit

适合做园林围墙、塔楼

下载方式: 下载 .zip → 解压 → models/ 文件夹里的 .glb

---

### 4. Sketchfab — 大量中国风模型（筛选可下载）
网址: https://sketchfab.com/search?q=chinese+pavilion&type=models&downloadable=true

搜索关键词: "chinese pavilion", "chinese garden", "moon gate", "pagoda"

注意: 只下载标注 "Downloadable" 且有 "CC Attribution" 或 "CC0" 许可的
