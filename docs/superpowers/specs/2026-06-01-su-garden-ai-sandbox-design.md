## 界面布局

三栏式 CSS Grid 布局：

```
┌──────────────┬──────────────────────────┐
│  左侧 40%    │  右上 60%×60%            │
│  鸟瞰地图    │  角色选择器 + 感知/对话流  │
│  (Canvas 2D) │                          │
│              ├──────────────────────────┤
│              │  右下 60%×40%            │
│              │  角色 POV 3D 视角        │
└──────────────┴──────────────────────────┘
```

- **左栏**：Canvas 2D 鸟瞰地图，显示园林布局 + 角色位置（彩色圆点），点击选中角色，右键标注目的地
- **右上栏**：角色切换标签 + 滚动文字流，展示选中角色的景色感受 & 与其他角色的对话
- **右下栏**：Three.js 渲染的 3D 画面，以选中角色的第一人称视角看园林

## 角色建模

极简几何体，展演时观众可远距离识别：

```
     ●  球体（头，radius ≈ 0.3）
    /|\
   / | \  四棱锥（身体，height ≈ 0.8）
  /__|__\

颜色方案：
  瓦工 = #B75C3A（赭石色 / 砖土）
  木匠 = #C4956A（木色）
  画师 = #6B8E7A（青灰色 / 水墨）
  自定义 = 用户任选
```

角色在 3D 场景中以 1.2 单位高度呈现，体型小巧，类似棋盘上的彩色棋子。

## 角色行为系统

```
角色状态机：
  wander ──→ observe ──→ converse
    │           │            │
    └───────────┴────────────┘
         （自动循环）

wander:   随机方向移动，触碰场景边界反弹
navigate: 用户右键标注目的地 → 角色平滑走向目标
observe:  每 8~12 秒触发，调用 LLM API 生成"看到景色"的感受
converse: 两角色距离 < 阈值（约 1.5 单位），调用 LLM API 生成对话
```

角色对象结构：
```
{
  id, name, role, color,
  personality: "香山帮瓦工，四十岁，专注飞檐与瓦当...",
  position: {x, y, z},
  rotation,
  target: null | {x, z},
  state: 'wander',
  observeCooldown: 8~12s,
  mesh: THREE.Group
}
```

## 对话系统

### 对话气泡（3D 场景内）

角色相遇时，头顶上方浮出对话气泡：

```
        "这飞檐的弧度…"
  ┌─────────────────┐
  │ 瓦工              │
  └────────┬────────┘
    ●瓦工   │ 1.5m  ●木匠
      ┌─────┴─────────┐
      │ 香山帮榫卯     │
      │ 不用一颗钉     │
      └───────────────┘
```

气泡用 CSS 定位覆盖在 3D 画面上（CSS2DRenderer 或 HTML overlay），2~3 秒后自动消失。

### 对话文字流（右上角）

同步显示完整的文字记录，带动画滚动效果，方便展演观众阅读。

### LLM Prompt 模板

```
感受生成：
"你是一个{role}，名叫{name}，性格{personality}。
你正在苏州园林中漫步，眼前看到的是{scene_description}。
请用第一人称，用口语化的中文，说一句你看到这个景色的感受。
一句话即可，不超过40字。"

对话生成：
"{角色A}和{角色B}在苏州园林中相遇。
{角色A}是{role_a}，{角色B}是{role_b}。
请生成一段简短的自然对话（2-3轮），
内容围绕香山帮营造技艺或苏州园林美学展开。"
```

## 场景素材

### 天空盒

使用苏州园林全景照片（等距矩形投影 / 6 面 Cubemap），效果类似置身园林。

数据来源（待调研）：
- 苏周到 APP 内"AI 云游"功能
- 公开的全景照片资源
- 自摄或购买

Demo 阶段使用临时占位天空盒（纯色渐变或公共领域全景图）。

### 鸟瞰地图

Demo 阶段用 Canvas 手绘园林平面示意（围墙、水面、建筑轮廓、路径），角色以彩色圆点标出。

## API 接入

- LLM API：OpenAI-compatible 接口（可使用 OpenAI / DeepSeek 等）
- API Key 在 Demo 阶段以变量形式存入代码（后续可加后端代理）
- 请求频率控制：感受 8~12s/次，对话按相遇触发
